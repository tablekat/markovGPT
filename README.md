# MarkovGPT

MarkovGPT! Who knows what the GPT stands for, don't worry about it.

## Development

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Getting Started

2. Install dependencies

   ```
   npm install
   # or
   yarn install
   ```

3. Start the development server

   ```
   npm run dev
   # or
   yarn dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser
